xSymmetrize {xUCINET}R Documentation

Symmetrizes the ties of a network

Description

This function allows different forms of symmetrization on a network.

Usage

xSymmetrize(NET1, Type = "Min")

Arguments

NET1

A network to be symmetrized.

Type

The type of symmetrization. The five options are: "Min" (take the minimum of both ties), "Max" (take the maximum of both ties), "Av" (take the average of both ties), "Sum" (take the sum of both ties), "Prod" (take the product of both ties).

Value

The symmetrized network.

Examples

M1<-matrix(c(1,2,3,4,5,6,7,8,NA),3,3)
xSymmetrize(M1)
