xTranspose {xUCINET}R Documentation

Transposes the ties of a network

Description

This function transposes a network containing network data.

Usage

xTranspose(NET1)

Arguments

NET1

A network to be transposed.

Value

The transposed network.

Examples

M1<-matrix(c(1,2,3,4,5,6,7,8,NA),3,3)
xTranspose(M1)
