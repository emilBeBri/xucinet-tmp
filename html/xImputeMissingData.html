xImputeMissingData {xUCINET}R Documentation

Impute missing data

Description

This function imputes missing data in a networks.

Usage

xImputeMissingData(NET1, Type = "Density", Loops = FALSE)

Arguments

NET1

The network to be used.

Type

The way to impute missing data. The current option is: "Density" (take the observed density as a basis).

Loops

Whether to include self-nominations.

Value

An imputed network.

Examples

M1<-matrix(c(0,NA,1,1,0,0,0,NA,0),3,3)
xImputeMissingData(M1, Type="Density")
