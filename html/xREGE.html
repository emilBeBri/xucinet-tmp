xREGE {xUCINET}R Documentation

Performs a REGE solution

Description

Takes a network as matrix or list of networks and finds the REGE solution.

Usage

xREGE(NET1)

Arguments

NET1

A network stored as a matrix or list of matrices

Value

A matrix representing regular equivalence values.

               0,0,0,0,1,
