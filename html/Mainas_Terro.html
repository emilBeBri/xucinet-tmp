Mainas_Terro {xUCINET}R Documentation

Contacts among people collected as part of counter-terrorism/Europol.

Description

Data obtained by Efstathios Mainas from Europol. Were collected and analyzed by both the Counter-Terrorism Unit and the Analysis Unit of Europol. Edges are the known contacts of the actors - the specific type of linkage was not revealed. See $ProjectInfo for more information.

Usage

Mainas_Terro

Format

An object of class list of length 4.
