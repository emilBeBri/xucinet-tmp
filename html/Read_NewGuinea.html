Read_NewGuinea {xUCINET}R Documentation

Network ties among Gahuku-Gama sub-tribes

Description

This data contain the social relations between Gahuku-Gama sub-tribes in NewGuinea as reported in Read (1954). Political (alliance) ties and opposition ties are reported. See $ProjectInfo for more information.

Usage

Read_NewGuinea

Format

An object of class list of length 5.
